1.git clone <url>
    downloads the repository
2.git status
    to see the changes
3.git add <filename>
    to move the changes from working directory to staging area
4.git add .
    to move all the changes
5.git commit -m "message"
    to move files from staging area to repository area
6.git push
    to push the changes from you localmachine(repository area) to github
7.git log
    to see the history
